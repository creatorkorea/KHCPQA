# Open Questions

사용자가 놓치기 쉬운 결정 사항과 개발 전 확인이 필요한 항목이다.

## 반드시 확인할 사항

1. 최종 브랜드명  
   KHCPQA, 한국건강관리사자격협회, SMC아카데미를 어떻게 병기할지 확정해야 한다.

2. 도메인  
   신규 도메인을 사용할지, 기존 `smc365.ac` 하위 또는 대체 도메인을 사용할지 확정해야 한다.

3. 콘텐츠 권리  
   기존 사이트의 사진, 후기, 보도자료, 강사 프로필 이미지를 새 사이트와 해외 언어 페이지에 사용해도 되는지 확인해야 한다.

4. 언어 정책  
   결정 완료: 한국어, 영어, 스페인어 3개 국어를 1차 정식 공개 언어로 운영한다.

5. 번역 책임  
   영어/스페인어 번역 초안 작성 후 누가 최종 감수할지 정해야 한다.

6. 자격 데이터 형식  
   자격번호, 발급일, 과정명, 사용자명, 상태를 어떤 엑셀/DB 형식으로 받을지 정해야 한다.

7. 공개 자격 검증 여부  
   로그인 없이 검증 코드로 유효/무효만 확인하는 공개 검증 기능을 넣을지 결정해야 한다.

8. 회원가입 방식  
   결정 완료: 누구나 회원가입할 수 있는 공개 가입 방식으로 운영한다.

9. 관리자 계정 정책  
   초기 관리자 수, 권한 구분, 비밀번호 정책을 정해야 한다.

10. 이미지 저장 방식  
   기존 이미지 URL을 참조할지, 새 스토리지로 복사할지 결정해야 한다.
