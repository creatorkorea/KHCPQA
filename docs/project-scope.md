# Project Scope

이 문서는 1차 개발 범위와 제외 범위를 정의한다.

## 포함 범위

- 반응형 홈페이지
- 모바일 우선 UX
- 화이트 + 그린 기관형 디자인 시스템
- 한국어, 영어, 스페인어 3개 국어 페이지 구성
- 협회 소개 페이지
- 과정 목록 및 과정 상세 템플릿
- 커뮤니티 목록 및 상세 템플릿
- 회원가입
- 로그인
- 비밀번호 찾기
- 회원정보 수정
- 로그인 후 My Page
- 자격 취득 조회
- 관리자 페이지
- 공지사항 관리
- 문의 관리
- 팝업/배너 관리
- 회원 관리
- 게시글 관리
- 자격 데이터 관리
- 호스팅 환경 배포 및 검수 URL 제공

## 제외 범위

아래 항목은 1차 개발 범위에서 제외한다.

- PG 결제
- 문자/카카오 알림
- 실시간 채팅
- 외부 CRM/ERP 연동
- 네이티브 iOS/Android 앱
- 앱스토어 등록
- 푸시 알림
- 고급 통계 대시보드
- GA4/Search Console 운영 대행
- 대량 데이터 정제
- 엑셀 업로드 자동화
- 기존 시스템 마이그레이션 자동화
- 전문 법률 검토
- 개인정보 영향평가
- 보안 컨설팅
- 촬영, 영상 제작
- 유료 이미지, 폰트, 플러그인 구매
- 증명서 자동 발급
- 자동 번역 시스템
- 전문 번역 감수

## 범위 판단 기준

- 계약서 별첨 A, B, C, D에 명시되지 않은 기능은 별도 범위로 본다.
- 블루프린트의 장기 로드맵은 참고 자료이며 1차 납품 범위가 아니다.
- 783개 수준의 상세 페이지는 전체 수동 정리보다 CMS 구조와 우선 콘텐츠 이관을 먼저 완료한다.
- 유지보수는 제작 계약과 별도다.

## 필수 납품 기준

- 공개 사이트 주요 페이지
- 로그인 및 계정 메뉴
- My Page 보호 라우트
- 자격 조회 샘플 플로우
- 관리자 기본 CRUD
- 기존 콘텐츠 기반 주요 페이지
- 반응형 검수
- 다국어 라우팅
- 배포 가능한 검수 URL
