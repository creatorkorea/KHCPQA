# Client Action Items

이 문서는 검수 URL 전달과 정식 오픈 전에 발주사 또는 운영자가 확정해야 하는 항목을 실행 가능한 요청 단위로 정리한다.

## 1. 계약/범위

| 요청 항목 | 필요한 답변/파일 | 완료 기준 |
| --- | --- | --- |
| 계약 범위 확인 | 현재 구현 범위 승인 또는 수정 요청 목록 | 구현 범위와 제외 범위가 문서로 확정됨 |
| 제외 범위 확인 | 1차 오픈 제외 기능 목록 | 오픈 후 후속 단계로 넘길 항목이 합의됨 |
| 최종 브랜드 표기 | KHCPQA, 한국건강관리사자격협회, SMC아카데미 병기 규칙 | 헤더, footer, SEO, 문서에 같은 표기 적용 |

## 2. 도메인/운영 계정

| 요청 항목 | 필요한 답변/파일 | 완료 기준 |
| --- | --- | --- |
| 도메인 결정 | 사용할 최종 도메인 또는 검수용 도메인 | Vercel/Supabase Auth URL에 반영 가능 |
| 호스팅 계정 명의 | Vercel/GitHub/Supabase 운영 계정 담당자 | 배포와 권한 이관이 가능 |
| SSL/검색 등록 | 운영 도메인 기준 SSL, Google/Naver Search Console 담당자 | `NEXT_PUBLIC_SITE_URL`과 검증 코드 확정 |

## 3. 콘텐츠 권리/출처

| 요청 항목 | 필요한 답변/파일 | 완료 기준 |
| --- | --- | --- |
| SMC365 콘텐츠 사용 권리 | 이미지, 후기, 보도자료, 강사 사진 사용 가능 여부 | 공개 사용 가능/불가 콘텐츠 구분 |
| 전체 source URL | 게시글/후기/사진/보도자료별 원본 URL 목록 | CMS 등록 콘텐츠의 `source_url` 누락 0건 |
| 로고 원본 | SVG, PNG, AI 등 사용 가능한 로고 파일 | 헤더/footer/OG 이미지에 최종 로고 적용 |

## 4. 법무/정책

| 요청 항목 | 필요한 답변/파일 | 완료 기준 |
| --- | --- | --- |
| 개인정보처리방침 | 최종 원문 | `/[locale]/privacy`에 실제 원문 반영 |
| 이용약관 | 최종 원문 | `/[locale]/terms`에 실제 원문 반영 |
| 자격 정보 공개 범위 | 로그인 회원 전용/공개 검증 여부 | 자격 조회 정책과 화면 문구 확정 |

## 5. 운영 데이터

| 요청 항목 | 필요한 답변/파일 | 완료 기준 |
| --- | --- | --- |
| 자격 데이터 샘플 | CSV 또는 엑셀: 이메일, 과정명, 자격번호, 발급일, 상태, 검증 코드 | Supabase `certifications` 샘플 import 가능 |
| 초기 관리자 목록 | 이메일, 이름, 역할, 활성 상태 | 관리자 권한 부여 가능 |
| 문의 처리 정책 | 상태값, 담당자 메모, 응답 SLA | 관리자 문의 관리 운영 규칙 확정 |

- `docs/templates/content-source-url-template.csv`
