# Scope Confirmation

이 문서는 1차 개발 범위와 제외 범위를 발주사 확인용으로 요약한다. 기준 문서는 `docs/project-scope.md`, `KHCPQA_WEBAPP_DEVELOPMENT_PLAN.md`, `agentd.md`다.

## 1차 포함 범위

| 구분 | 포함 내용 | 현재 상태 |
| --- | --- | --- |
| 공개 사이트 | 홈, 협회 소개, 과정 목록/상세, 활동/커뮤니티, Contact | 구현 및 QA 완료 |
| 다국어 | 한국어, 영어, 스페인어 라우팅과 언어 선택 | 구현 및 QA 완료 |
| 회원 기능 | 회원가입, 로그인, 비밀번호 찾기, My Page, Profile | 구현 및 E2E 완료 |
| 자격 조회 | 로그인 회원의 자격 목록과 검증 코드 조회 | 구현 및 E2E 완료 |
| 문의 | 파트너 문의 제출과 관리자 문의함 저장 | 구현 및 E2E 완료 |
| 관리자 | 콘텐츠, 문의, 자격 데이터, 배너, 회원 권한 관리 | 구현 완료 |
| 보안 | 관리자/My Page 보호 라우트, noindex, RLS | 구현 및 검증 완료 |
| 반응형 | 모바일 375px, 태블릿 768px, 데스크톱 | QA 완료 |
| 배포 준비 | GitHub 최신화, Vercel 인계 문서, 환경변수 목록 | 문서화 완료 |

## 1차 제외 범위

| 구분 | 제외 내용 | 후속 처리 |
| --- | --- | --- |
| 결제 | PG 결제, 유료 수강 결제 | 별도 견적 |
| 알림 | 문자, 카카오, 이메일 자동 알림 | 별도 유지보수/후속 개발 |
| 앱 | 네이티브 iOS/Android 앱, 앱스토어 등록 | 별도 프로젝트 |
| 자동화 | 엑셀 업로드 자동화, 기존 시스템 자동 마이그레이션 | 실데이터 확보 후 후속 개발 |
| 고급 기능 | 고급 통계, 검색 고도화, 증명서 자동 발급 | 후속 단계 후보 |
| 법무/보안 | 전문 법률 검토, 개인정보 영향평가, 보안 컨설팅 | 발주사 또는 전문 업체 확인 |
| 제작 자산 | 촬영, 영상 제작, 유료 이미지/폰트 구매 | 발주사 제공 또는 별도 비용 |
| 번역 | 전문 번역 감수, 자동 번역 시스템 | 발주사 최종 감수 필요 |

## 발주사 확인 요청

- 위 포함 범위가 1차 검수 기준으로 맞는지 확인
- 위 제외 범위가 1차 오픈 제외 항목으로 맞는지 확인
- 추가 요청이 있을 경우 계약 범위 내 오류/누락인지, 후속 개발인지 구분

## 현재 판단

개발 관점의 1차 범위는 완료 상태다. 남은 항목은 도메인, 법무 원문, 콘텐츠 권리, 실운영 데이터, 전체 `source_url` 확정, 검수 URL 전달처럼 발주사 또는 운영자 의사결정이 필요한 항목이다.
